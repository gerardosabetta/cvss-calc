# CVSS 3.1 Calculator

A simple, stylish web-based calculator for the **Common Vulnerability Scoring System v3.1**. Select values for each base metric and get a real-time CVSS score, severity rating, and vector string.

## Features

- All 8 CVSS 3.1 base metrics: Attack Vector, Attack Complexity, Privileges Required, User Interaction, Scope, Confidentiality, Integrity, and Availability
- Real-time score calculation following the official CVSS v3.1 specification
- Color-coded severity display (None, Low, Medium, High, Critical)
- Live vector string generation
- Toggle selections on/off, reset all with one click

## Tech Stack

- React + TypeScript
- Vite
- Inline CSS (no external stylesheets or CSS-in-JS libraries)

## Getting Started

```bash
npm install
npm run dev
```

## Build

